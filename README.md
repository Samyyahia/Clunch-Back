Clunch Back Office
========================

Symfony 3.4 Project

Dependencies
--------------

  * **Basic Symfony v3.4 Setup**
  * **FOSUserBundle v2.0**
  * **Sonata v3.35**

--------------

Setup
--------------

### Install
```shell
  $ composer install
  $ php bin/console assets:install
  $ php bin/console doctrine:schema:update --force
```

### Launch
```shell
  $ php bin/console server:run
```
